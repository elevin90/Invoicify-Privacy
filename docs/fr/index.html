Politique de confidentialité

1. Introduction

Invoicify ("l'App") respecte votre vie privée. Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons et quels sont vos droits concernant ces données.

2. Collecte et utilisation des données

Nous collectons et traitons les données suivantes :

2.1 Données utilisateur (stockées dans iCloud via CloudKit)

L'App stocke les factures et les informations des clients dans CloudKit d’Apple pour synchroniser les données entre vos appareils Apple. Ces données sont privées et accessibles uniquement par vous. Nous n’avons pas accès à ces données.

2.2 Analyse et rapports d’erreurs (Firebase)

Nous utilisons Google Firebase pour les analyses de base et les rapports de crash. Firebase peut collecter des données d’utilisation anonymisées afin d'améliorer l'App. La collecte de données par Firebase est régie par la politique de confidentialité de Google.

2.3 Données d'abonnement (StoreKit)

L'App propose des abonnements premium via StoreKit d’Apple. Le traitement des paiements et la gestion des abonnements sont pris en charge par Apple. Nous ne collectons ni ne stockons les détails de paiement. Pour plus d’informations, veuillez consulter la politique de confidentialité d’Apple.

3. Partage des données

Nous ne vendons, ne louons et ne partageons pas vos données avec des tiers. Vos données sont stockées uniquement sur votre appareil et dans votre compte iCloud.

4. Conservation et suppression des données

Vos données sont stockées dans iCloud tant que l'App est installée. Vous pouvez supprimer toutes les données stockées en :

  • Supprimant l'App de votre appareil (supprime les données locales).
  • Supprimant les données de l'App d’iCloud dans les réglages Apple.
  • Utilisant l’option "Réinitialiser les données" dans les paramètres de l’App (efface toutes les données stockées).

5. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Toute modification sera reflétée dans la dernière version de l'App.
