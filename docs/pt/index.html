Política de Privacidade

1. Introdução

O Invoicify ("o Aplicativo") respeita sua privacidade. Esta Política de Privacidade explica quais dados coletamos, como os usamos e quais são seus direitos sobre esses dados.

2. Coleta e uso de dados

Coletamos e processamos os seguintes dados:

2.1 Dados do usuário (armazenados no iCloud via CloudKit)

O Aplicativo armazena faturas e informações de clientes no CloudKit da Apple para sincronizar os dados entre seus dispositivos Apple. Esses dados são privados e acessíveis apenas por você. Não temos acesso a esses dados.

2.2 Análise e relatórios de falhas (Firebase)

Usamos o Google Firebase para análises básicas e relatórios de falhas. O Firebase pode coletar dados anônimos de uso para ajudar a melhorar o Aplicativo. A coleta de dados do Firebase é regida pela Política de Privacidade do Google.

2.3 Dados de assinatura (StoreKit)

O Aplicativo oferece assinaturas premium por meio do StoreKit da Apple. O processamento de pagamentos e a gestão de assinaturas são feitos pela Apple. Não coletamos nem armazenamos detalhes de pagamento. Para mais informações, consulte a Política de Privacidade da Apple.

3. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados com terceiros. Seus dados são armazenados apenas no seu dispositivo e na sua conta do iCloud.

4. Retenção e exclusão de dados

Seus dados permanecem armazenados no iCloud enquanto o Aplicativo estiver instalado. Você pode excluir todos os dados armazenados ao:

  • Remover o Aplicativo do seu dispositivo (remove os dados locais).
  • Excluir os dados do Aplicativo no iCloud nas configurações da Apple.
  • Usar a opção "Redefinir Dados" nas configurações do Aplicativo (apaga todos os dados armazenados).

5. Alterações nesta política

Podemos atualizar esta Política de Privacidade. Quaisquer alterações serão refletidas na versão mais recente do Aplicativo.
